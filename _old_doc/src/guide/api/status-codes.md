# Status Codes

| Method |              Description               |
| ------ | :------------------------------------: |
| 200    |         OK - request succeeded         |
| 400    |    Bad Request - malformed request     |
| 403    | Forbidden - the request is not allowed |
| 404    |  Not Found - resource does not exist   |
| 5xx    |              Server Error              |

::: warning
We are still a beta version
:::
